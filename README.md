grid-mesh
=========
Creates a grid mesh

Usage
=====
Install using npm:

    npm install grid-mesh
    
And use as follows:

```javascript
console.log(require("grid-mesh")(100, 100))
```

`require("grid-mesh")(nx, ny)`
------------------------------
Creates an nx by ny sized grid mesh.
